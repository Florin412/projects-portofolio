MobiExchange App

The currency exchange application developed during the internship is a full-stack project, including a front-end in React and a back-end in Java and Spring. The front-end comprises login, registration, and other pages, while the back-end provides numerous APIs documented with Swagger-UI.

The main features include token-based authentication, a currency conversion calculator, and a chart for the evolution of currency prices.
